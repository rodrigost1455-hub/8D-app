# 8D Quality Management System

Web application for managing 8D (Eight Disciplines) problem-solving workflows 
in automotive supplier ecosystems. Built to replace spreadsheet-based 
non-conformance tracking with a structured, traceable digital process.

## The Problem

In automotive Tier-1 manufacturing, 8D responses to customer complaints are 
typically managed through email chains and Excel files. This creates:
- No single source of truth for corrective action status
- No audit trail for IATF 16949 compliance reviews
- Manual tracking of deadlines and responsible owners
- No visibility across multiple concurrent 8Ds

This app centralizes the entire 8D workflow — from D1 (team formation) 
to D8 (closure) — in a single web interface.

## Features

- Full 8D workflow management (D1 through D8)
- Non-conformance creation and tracking
- Corrective action assignment and deadline management
- Status visibility across the supplier ecosystem
- Audit-ready documentation trail
